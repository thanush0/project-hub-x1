Example Movie Caption Format for Telegram Uploads
===================================================

Copy and paste this template when uploading movies to your Telegram channel.
Replace the values with your movie's actual information.

===================================================
TEMPLATE
===================================================

Title: Movie Title Here
Year: 2024
Genre: Action, Thriller, Drama
Rating: 8.5
Description: Brief description of the movie plot. You can write multiple lines here to provide more details about the storyline, cast, and any other relevant information.

===================================================
REAL EXAMPLES
===================================================

Year: 2015
Genre: Action, Adventure, Sci-Fi

Description: In a post-apocalyptic wasteland, a woman rebels against a tyrannical ruler in search for her homeland with the aid of a group of female prisoners, a psychotic worshiper, and a drifter named Max.

===================================================
FIELD EXPLANATIONS
===================================================

Title:       The name of the movie/show (required)
Year:        Release year in YYYY format (required)
Genre:       One or more genres, separated by commas (required)
Rating:      IMDb rating, Rotten Tomatoes, or age rating (optional)
Description: Plot summary - can be multiple lines (optional)

===================================================
TIPS
===================================================

1. Keep title concise and accurate
2. Use comma-separated genres (no "and")
3. Rating can be IMDb score (8.5) or age rating (PG-13)
4. Description should be engaging but spoiler-free
5. For series, include season and episode: "S01E01 - Episode Title"
6. Use consistent formatting across all uploads
7. You can add custom fields if needed

===================================================
MINIMAL CAPTION (If You're Lazy)
===================================================

Title: Movie Name
Year: 2024
Genre: Action
Rating: N/A
Description: No description available.

Note: The app will use the filename if no caption is provided,
      but having proper metadata makes browsing much better!
