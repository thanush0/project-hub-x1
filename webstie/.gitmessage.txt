feat(functions): add multi-provider email service and wire up contact/custom request handlers

Implements a provider-agnostic email service with SendGrid, Mailgun, and SMTP support, including development-mode fallback logging. Updates Netlify serverless functions to send admin notifications and client confirmations.

- Email service (netlify/functions/services/email-service.ts)
  - Providers: sendgrid | mailgun | smtp | development
  - Helpers: formatContactFormEmail, formatCustomRequestEmail, formatClientConfirmationEmail
- contact-form.ts
  - Validates input and email, logs submissions
  - Sends admin notification (reply-to = user email)
- custom-request.ts
  - Generates requestId, logs submissions
  - Sends admin notification + client confirmation
  - Optional SAVE_REQUESTS_TO_GIT support (content/requests)
- Env + docs
  - project-hub-x/.env.example enriched with email vars
  - EMAIL_SETUP_GUIDE.md with provider setup + testing
  - README and DEPLOYMENT_GUIDE pointers
- Tooling
  - tools/test-email-functions.mjs end-to-end tests
  - tools/test-email-service-local.mjs guidance script
